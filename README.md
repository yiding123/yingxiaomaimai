# yingxiaomaimai

营销脉脉是个微信小程序，主要目的是做营销方面的资源整合。
营销脉脉后端服务使用的应用窝，在开发进速度上大大提高，只用一个前端工程师就可以完成整个应用，后端服务全部由"应用窝www.cloudadmx.com"  配置生成，也免去了前端工程师与后端工程师在接口服务上的协作。最关键的是连服务器也都不用租用了，"应用窝www.cloudadmx.com"提供了数据服务，还提供了https的域名，非常方便。
